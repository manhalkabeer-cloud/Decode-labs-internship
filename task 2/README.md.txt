# Task 2 - Sensor Data Simulation

## Objective

The objective of this task is to simulate environmental sensor data using Python and store the generated values in a CSV file.

## Features

* Random temperature generation
* Random humidity generation
* Motion detection simulation
* Automatic CSV file creation

## Files

* sensor_simulation.py
* sensor_data.csv

## Software Used

* Python
* Pandas

## How to Run

```bash
python sensor_simulation.py
```

## Expected Output

The program generates random sensor values and saves them into **sensor_data.csv**.

## Conclusion

This task demonstrates the basic concept of IoT sensor simulation and data storage using Python.
