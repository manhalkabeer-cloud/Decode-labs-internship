# Decode-labs-internship

## About

This repository contains the tasks completed during my Internet of Things (IoT) Internship at DecodeLabs.

## Completed Tasks

### Task 2 – Sensor Data Simulation
- Simulated temperature, humidity, and motion sensor values.
- Stored sensor readings in a CSV file.
- Displayed sensor data using Python.

### Task 4 – IoT Automation Logic
- Implemented automation rules based on sensor values.
- Simulated the operation of a fan, dehumidifier, and security alarm.

### Task 5 – Smart Environment Monitoring System
- Combined sensor simulation and automation.
- Built a Streamlit dashboard for monitoring sensor values.

## Technologies Used
